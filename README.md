# Snake Game (Python)

A simple Snake game built with Python.

## Requirements

- Python 3.10 or newer
- `tkinter` (usually included with Python)

## Run

bash
python snake_game.py
Controls:-
Arrow keys or W A S D to move
Space to restart after game over
